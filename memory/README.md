# momo 轻量记忆

这个目录存放 `momo` 的轻量记忆，不包含复杂长期代理逻辑。

## 结构

- `facts.jsonl`
  - 稳定事实
  - 例如用户偏好、称呼、明确要求记住的事项
- `profile.json`
  - 聚合后的长期偏好快照
  - 便于运行时直接读取，而不是每次重扫全部 facts
  - 会处理去重、较新偏好覆盖、以及显式冲突裁决
- `daily/YYYY-MM-DD.md`
  - 每天的互动记录
  - 包括用户说了什么、执行是否成功、关键结果
- `heartbeat/latest.md`
  - 最近一次 heartbeat 自检摘要

## 设计原则

- 先做轻量、可读、可维护
- 优先保留原始记录，不做过度智能化猜测
- 只有较明显的稳定信息才写入 `facts.jsonl`
- `profile.json` 只做聚合视图，不替代原始事实日志
- 如果出现冲突偏好，优先保留更高置信度或更新的事实，并在 `profile.json` 里留下冲突记录
